# Portfólio Renato Herisson

Portfolio website profissional com design minimalista (tema cinza claro #f5f5f5) e formulário de contato.

## 🚀 Como Rodar

### 1. Instalar Dependências
```bash
npm install
```

### 2. Iniciar o Servidor
```bash
npm start
```
Acesso: `http://localhost:3000`
